# SilaseZ Profile

Landing page resmi SilaseZ dan halaman unduh aplikasi Android.

## Menjalankan secara lokal

Buka `index.html` langsung di browser atau jalankan static HTTP server dari direktori ini.

## Rilis aplikasi

Tombol unduh mengarah ke aset `SilaseZ-v1.0.4.apk` pada GitHub Release terbaru.

## Teknologi

HTML, CSS, dan JavaScript tanpa framework atau layanan analitik eksternal.
